# OrderManager - Sistema de Processamento de Pedidos

Este projeto consiste em uma Web API desenvolvida em **.NET 8** focada no processamento de pedidos com cálculo de impostos dinâmico via Feature Flags, seguindo os princípios de **Clean Architecture** e **DDD (Domain-Driven Design)**.

## 🚀 Como Executar o Projeto

### Pré-requisitos
* [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
* Visual Studio 2022 ou VS Code

### Passo a Passo
1. **Clonar o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/ordermanager.git](https://github.com/seu-usuario/ordermanager.git)
   cd ordermanager

2.Restaurar dependências:
dotnet restore

3.Executar a API:
dotnet run --project OrderManager.API
